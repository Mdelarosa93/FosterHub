# FosterHub

**Simplifying Foster Care, Empowering Families**

FosterHub is a browser-first platform concept for improving foster care operations across DHR staff, caseworkers, foster/resource parents, vendors, and other approved participants.

This repository currently contains the early product foundation for FosterHub, including:
- product planning documents
- technical architecture drafts
- permissions and portal structure
- database and API drafts
- MVP roadmap and sprint planning
- QA and Jira-ready planning artifacts
- a clickable prototype

## Project Status

FosterHub is currently in the **product definition + prototype** stage.

The repository is being used to shape:
- product strategy
- user roles and permissions
- portal structure
- core workflows
- MVP delivery scope
- technical implementation direction

## Clickable Prototype

The `prototype-v1/` folder contains a lightweight browser prototype used to explore the product visually.

A GitHub Pages-ready copy is also stored in `docs/`, so the prototype can be published directly from the repository.

Current prototype coverage includes:
- Caseworker Portal
- DHR Admin Portal
- Resource Parent Portal
- Vendor Portal

### Main screens included so far
- worker dashboard
- case list
- case detail
- request queue
- placement search
- admin dashboard
- intake queue
- vendor review queue
- permissions view
- parent dashboard
- parent child detail
- approved vendors
- vendor onboarding
