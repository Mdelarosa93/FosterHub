# FosterHub App Monorepo

This is the implementation scaffold for FosterHub.

## Apps

- `apps/web` - Next.js frontend for portals and public pages
- `apps/api` - NestJS backend API
- `apps/worker` - background jobs and async workflows

## Packages

- `packages/ui` - shared UI components
- `packages/types` - shared types
- `packages/auth` - shared auth utilities and permission helpers

## Planned MVP build order

1. Auth + permissions foundation
2. Child/case intake flow
3. Case list and case detail
4. Worker dashboard
5. Resource parent portal basics
6. Requests and approvals
7. Placement workflows
8. Vendor onboarding basics
